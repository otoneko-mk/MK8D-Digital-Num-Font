# MK8D-Digital-Num-Font

マリオカート8デラックスのデジタル数字風のフォントです

任天堂株式会社及び開発関係者は一切関与しておりません

| フォント名         | フォントファイル                                                                                               | 説明                                                     |
|----------------|--------------------------------------------------------------------------------------------------------|--------------------------------------------------------|
| MK8DDigitalNum | [MK8DDigitalNum.ttf](https://github.com/otoneko-mk/MK8D-Digital-Num-Font/blob/main/MK8DDigitalNum.ttf) | マリオカート8デラックスのデジタル数字風のフォントです<br>ポイントやレートの表示にお使いください |
